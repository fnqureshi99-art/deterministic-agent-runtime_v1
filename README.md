# Deterministic Agent Runtime

An advanced, open-source runtime environment for building, orchestrating, and executing deterministic AI agents. Designed for reliability, privacy, and seamless enterprise integration.

![Deterministic Agent Runtime UI](https://picsum.photos/seed/deterministic/1200/600)

## 🚀 Features

- **Multi-Provider LLM Support:** Seamlessly switch between Google Gemini (Default), OpenAI, Anthropic, Groq, and Local models (Ollama/LMStudio) for privacy-first execution.
- **Deterministic Orchestration:** Complex goals are automatically deconstructed into parallelizable, actionable tasks by a master planner agent.
- **Secure Local Sandboxing:** Execute Python code directly in the browser using Pyodide. No server-side execution required, ensuring maximum security and privacy for sensitive data.
- **Multimodal Capabilities:** Built-in support for vision (image uploads) and voice (TTS/STT) interactions.
- **Enterprise Integrations:** Native, user-configurable OAuth flows for Slack, Stripe, and Confluence. Bring your own Client IDs.
- **Artifact Generation:** Agents generate specific artifacts (Code, Python, Documents, Data) that can be reviewed, edited, and executed.

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite, Tailwind CSS v4, Framer Motion, Lucide React
- **Backend:** Node.js, Express (for OAuth callback handling)
- **AI/ML:** `@google/genai`, Pyodide (WebAssembly Python)
- **Styling:** Tailwind CSS with a clean, modern, "technical dashboard" aesthetic.

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-org/deterministic-agent-runtime.git
   cd deterministic-agent-runtime
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure Environment Variables:
   Copy the `.env.example` file to `.env` and add your API keys and OAuth Client IDs.
   ```bash
   cp .env.example .env
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000`.

## 🔐 Security & Privacy

The Deterministic Agent Runtime is built with privacy in mind. 
- **Local Execution:** Python code generated by agents is executed entirely within your browser's WebAssembly sandbox via Pyodide.
- **Local Storage:** API keys and OAuth credentials configured in the UI are stored locally in your browser's `localStorage`.
- **Local LLMs:** Support for local LLMs (like Ollama) means your data never has to leave your machine.

## 🔌 Integrations (OAuth)

To use the integrations (Slack, Stripe, Confluence), you need to configure OAuth apps in their respective developer portals.

1. Create an OAuth app in the provider's portal.
2. Set the redirect URI to `http://localhost:3000/auth/<provider>/callback` (or your production URL).
3. Enter the Client ID in the platform's Settings modal under "Integration Credentials".
